# GeneralizedTDGL_Oh
This repository contains Python code for numerical simulations of the induction of p-wave and lower-symmetry singlet order parameters in a centrosymmetric s-wave superconductor with O<sub>h</sub> group symmetry via electromagnetic radiation, e.g., a microwave or sub-THz beam, that was used for producing some of the data in Ref. [1].

It also contains movies showing the dynamics of order parameters for the cases of linearly and circularly polarized lights, frames from which were used in Figs. 5-12. Corresponding parameters for the simulation can be found in Ref. [1].

--References:

1. H. Yerzhakov and A. V. Balataksy. Induction of p-wave and d-wave order parameters in s-wave superconductors with light pulses. _To appear on arXiv soon._

## Repository structure

- `GeneralizedTDGL_Oh.py` – main Python implementation of the generalized TDGL simulations.
- `GenTDGL_exploratory.ipynb` – exploratory research notebook used during model development and numerical testing. It contains intermediate calculations, parameter exploration, and prototype implementations that supported the development of the main simulation code, but is not intended as a polished software package.
- `*.mp4` – representative simulation movies generated by the code.

# Install via pip
From this GitHub repository:

```bash
pip install git+https://github.com/hennadii-phys/GeneralizedTDGL_Oh.git
```

Editable installation:

```bash
git clone https://github.com/hennadii-phys/GeneralizedTDGL_Oh.git
cd GeneralizedTDGL_Oh
pip install -e ".[dev,docs]"
```
