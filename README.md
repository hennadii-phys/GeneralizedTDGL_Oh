# GeneralizedTDGL_Oh
This repository contains a Python code for numerical simulations of the induction of p-wave and lower-symmetry singlet order parameters in a centrosymmetric s-wave superconductor with Oh group symmetry via electromagnetic radiation, e.g., microwave or sub-THz beam, that was used for producing some of the data in Ref. [1].

--References:

1. Induction of p-wave and d-wave order parameters in s-wave superconductors with light pulses. _To appear on arXiv soon._

# Install via pip
From this GitHub repository:

pip install git+https://github.com/hennadii-phys/GeneralizedTDGL_Oh.git

Editable installation:

git clone https://github.com/hennadii-phys/GeneralizedTDGL_Oh.git

cd GeneralizedTDGL_Oh

pip install -e ".[dev,docs]"
