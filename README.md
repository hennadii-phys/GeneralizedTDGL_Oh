# GeneralizedTDGL_Oh
This is a Python code for numerical simulations of the induction of p-wave and lower-symmetry singlet order parameters in a centrosymmetric s-wave superconductor with Oh group symmetry via electromagnetic radiation, e.g., microwave or sub-THz beam.

# Install via pip
From this GitHub repository:

pip install git+https://github.com/hennadii-phys/GeneralizedTDGL_Oh.git

Editable installation:

git clone https://github.com/hennadii-phys/GeneralizedTDGL_Oh.git
cd lg-tdgl
pip install -e ".[dev,docs]"
